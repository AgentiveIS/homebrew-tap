# Agentive Homebrew Tap

Homebrew tap for [Agentive](https://agentive.is) products.

## Installation

```bash
brew tap AgentiveIS/tap
```

## Available Formulas

### MyAgentive

Open-source personal AI agent for power users.

```bash
brew install myagentive
```

**Documentation:** [github.com/AgentiveIS/MyAgentive](https://github.com/AgentiveIS/MyAgentive)
